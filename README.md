## Начало работы

Установите SDK:

```bash
pip install dawg-baas
```

Получите API-ключ в [личном кабинете](https://dawgswarm.ru/dashboard) и задайте переменную окружения:

```bash
export DAWG_API_KEY="your-api-key"
```

## Как добавить свой пример

1. Форкните репозиторий
2. Создайте папку в соответствующей категории: `examples/<категория>/<название>/`
3. Добавьте в папку:
   - Скрипт с примером
   - `README.md` с описанием: что делает, какие зависимости, как запустить
   - `requirements.txt` (если есть дополнительные зависимости)
4. Откройте Pull Request

### Требования к примерам

- Рабочий код на Python 3.10+
- Использование `dawg-baas` SDK
- Никаких захардкоженных API-ключей или credentials
- README с описанием и инструкцией по запуску
- Корректная обработка ошибок и таймаутов
- Разумные задержки между запросами
